General Game Instructions

Welcome to this experiment. Today you are going to play the Mirror Game. The Mirror Game is an improvisation game where the objective is for participants to synchronize between each other while creating a variety of movements. The movements simulate dancing and the game is meant to be played using creative movements that are done making new movements that use different trajectories within the game area.

You will play this game with a teammate. There will be a score linked to how much of the screen you use to create movements, how you and your teammate synchronize and complement each others' movements.
